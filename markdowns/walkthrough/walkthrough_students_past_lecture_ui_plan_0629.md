# 수강생 이전 강의(VOD) 블록 개편 완료 안내

Home 탭과 Classroom 탭의 강의 영상들이 매일 쌓이는 구조를 깔끔하게 관리할 수 있도록 주차별(Week) 그룹핑 및 화면 전환(Drill-down) 방식의 UI 개편을 모두 완료했습니다.

## 핵심 변경 내용

### 1. `WeeklyVODBlock` 공통 컴포넌트 개발
Home과 Classroom 양쪽에서 재사용 가능한 전용 컴포넌트를 만들었습니다.
- **주차별 그리드 뷰 (기본 화면)**
  - 가장 상위 뷰에서는 `[Week 1 강의]`, `[Week 2 강의]` 와 같이 주차별로 묶인 폴더형 썸네일 카드가 표시됩니다.
  - 이 카드의 썸네일은 **해당 주차에서 가장 최근(최신 날짜)에 업로드된 영상의 썸네일**로 자동 지정됩니다. 우측 상단에는 해당 주차에 몇 개의 영상이 있는지 개수 뱃지도 함께 뜹니다.
- **Drill-down 상세 리스트 뷰**
  - 특정 주차 카드를 터치(클릭)하면, 모바일 앱처럼 매끄럽게(Slide 애니메이션) 해당 주차의 개별 영상 리스트 뷰로 전환됩니다.
  - 상단의 "뒤로 가기" 버튼을 누르면 다시 전체 주차 그리드 뷰로 돌아옵니다.

### 2. "전체 보기" 버튼 추가
블록 우측 상단에 "전체 보기" 버튼을 추가했습니다. 추후 이 버튼을 누르면 모달을 띄우거나 전용 VOD 전체 리스트 페이지(`/student/lectures`)로 이동할 수 있도록 확장성을 마련해두었습니다.

### 3. 데이터 구조 및 더미 데이터 반영
- 데이터 모델(`RecordedLecture`)에 `week` 속성을 추가했습니다.
- `student.ts`의 더미 데이터를 확장하여 1주차 영상 2개, 2주차 영상 3개로 구성해 실제로 리스트가 어떻게 나타나는지 볼 수 있게 구성했습니다.

## 테스트 방법
1. 로컬 환경에서 `npm run dev` 를 실행해 주세요.
2. 수강생(Student)으로 로그인 후 메인 **Home 탭**에서 하단 `[이전 강의 보기]` 블록을 확인합니다.
3. 1주차 또는 2주차 썸네일 카드를 눌러 개별 영상 리스트가 잘 나오는지 테스트해 보시고, 다시 **"뒤로 가기"**가 작동하는지 확인해 보세요.
4. **Classroom 탭**에서도 동일한 UI 경험이 적용되었는지 확인하실 수 있습니다.

> [!TIP]
> 이제 매일 새로운 녹화 영상이 추가되더라도 `week` 데이터만 할당되면 가장 최신 영상의 썸네일을 대표로 하여 깔끔하게 주차별로 정돈됩니다. 추가로 변경하고 싶은 애니메이션이나 문구가 있다면 편하게 말씀해 주세요!
