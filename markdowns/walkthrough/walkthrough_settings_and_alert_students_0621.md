# 🔔 Notifications & Settings Implementation Walkthrough

수강생 분들이 모바일 PWA 환경에서 더욱 편리하게 알림을 확인하고 개인 설정을 관리할 수 있도록, **알림 센터**와 **설정 화면** 개편을 완료했습니다.

## 1. 알림 센터 (Notifications Center) 신규 구현
앱 상단의 헤더에 있는 **종 모양(Bell) 아이콘**을 클릭하면, 새롭게 만들어진 `/student/notifications` 페이지로 이동합니다.

*   **카테고리별 시각화:** 출석 경고(Warning), 공지사항(Info), 과제 마감(Assignment), 학습 자료(Material), 커뮤니티(Community) 등 알림 성격에 따라 아이콘과 색상이 다르게 표시되어 직관적으로 파악할 수 있습니다.
*   **읽지 않음 표시:** 아직 확인하지 않은 알림에는 우측 상단에 빨간색 점(Dot)이 표시되며, 상단 타이틀 옆에 읽지 않은 알림의 총 개수 뱃지가 뜹니다.
*   **모두 읽음 처리:** 상단의 `모두 읽음` 버튼을 누르면 읽지 않은 모든 알림이 한 번에 회색조(읽음 처리)로 바뀝니다.

## 2. 설정 화면 (Settings) 대폭 개편
기존에 스위치만 덩그러니 있던 설정 화면을 벗어나, 수강생 맞춤형 종합 환경 설정 화면으로 탈바꿈했습니다.

*   **내 프로필 확인:** 화면 최상단에 수강생의 아바타, 이름, 기수(예: AI 1기), 이메일 정보가 표시되는 프로필 카드가 추가되었습니다.
*   **세분화된 알림 제어:** `출석 및 학사 경고 알림`, `공지사항 및 학습 자료 알림`, `커뮤니티 활동 알림` 등을 각각 켜고 끌 수 있는 기능이 구현되었습니다.
*   **앱 환경 설정:** 다크 모드 토글과 첫 시작 화면 설정 탭을 추가했습니다.
*   **고객 지원 섹션:** `1:1 운영팀 문의`, `이용약관 및 정책`, `버전 정보`를 확인할 수 있는 메뉴를 구성하여 모바일 앱다운 사용성을 높였습니다.

> [!NOTE]
> 현재 두 페이지의 데이터(알림 내역, 사용자 정보, 켜짐/꺼짐 상태)는 시연을 위한 **임시 데이터(Mock Data)**로 구성되어 있습니다. 디자인과 기능 동작 확인용이며, 추후 백엔드 API 연동을 통해 실제 데이터 기반으로 작동하게 됩니다.

이제 상단 알림 아이콘과 하단 네비게이션의 설정 탭(기어 아이콘)을 눌러 새롭게 바뀐 UI를 직접 체험해 보세요!
