# 커뮤니티 글쓰기 및 파일 업로드 팝업 구현

## 작업 요약
수강생 커뮤니티 페이지(자유 게시판) 상단에 페이스북/인스타그램 같은 형태의 "새로운 글을 작성해보세요..." 버튼을 만들고, 이를 클릭했을 때 글과 파일을 첨부할 수 있는 팝업 모달을 완성했습니다.

## 주요 변경 사항

### 1. 피드 상단 글쓰기 UI 연동
- **[Community.tsx](file:///c:/Easyplex_AI/frontend/src/pages/student/Community.tsx)** 
- 스크롤 시 가장 잘 보이는 피드 최상단에 사용자 프로필 이미지와 입력창을 모방한 버튼 카드(`community-create-block`)를 배치했습니다. 

### 2. 글쓰기 팝업 (모달) 컴포넌트 추가
- **[CreatePostModal.tsx](file:///c:/Easyplex_AI/frontend/src/pages/student/components/CreatePostModal.tsx)** 신규 컴포넌트 추가.
- Tailwind CSS v4를 사용하여 화면 정중앙에 깔끔하게 나타나는 다이얼로그 모달을 디자인했습니다. (`scaleIn` 애니메이션 포함)
- 내용 입력란(textarea)과 파일 첨부 기능이 하단 툴바에 위치해있어 모바일에서도 글 작성이 편리합니다.

### 3. 파일 업로드 필터링 및 보안 로직
- 첨부 파일 버튼 `<input type="file" multiple>`을 활용하여 여러 개의 문서를 한 번에 올릴 수 있도록 구현했습니다.
- 브라우저 표준 `accept` 속성에 더하여 **JavaScript 내장 확장자 검사 로직**을 꼼꼼하게 추가했습니다.
  > [!IMPORTANT]
  > `.js`, `.py`, `.java`, `.exe`, `.sh` 등 시스템 보안 및 악성 스크립트 실행의 우려가 있는 **코딩/스크립트 파일 확장자는 즉시 차단**되며 브라우저 경고창(Alert)을 띄우고 파일 목록에서 자동 제외됩니다.
  > 그 이외의 이미지, 오디오, 동영상, 일반 문서는 첨부가 허용되어 첨부된 목록 배지로 나타납니다.

### 4. 실시간 피드 반영
- 글 내용을 입력하거나 파일을 첨부한 뒤 [게시] 버튼을 누르면, **작성한 게시글 내용(및 첨부파일 개수)이 즉시 커뮤니티 목록 최상단에 새롭게 등록**되도록 구현을 마쳤습니다.

## 확인 방법
1. 'Community' 탭으로 이동합니다.
2. 페이지 상단에 생긴 "새로운 글을 작성해보세요..." 버튼을 누릅니다.
3. 팝업창에서 임의의 글을 쓰거나 파일을 첨부해봅니다. (특히 `.py` 같은 파일 첨부 시도 시 어떻게 막히는지 확인해 보세요.)
4. "게시" 버튼을 누른 후 새 글이 바로 피드에 반영되는지 확인합니다.
